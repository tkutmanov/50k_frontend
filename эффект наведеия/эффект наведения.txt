Задача:

По наведению на элемент меню отобразить подробные данные

Решение

1) По наведению = :hover
2) на элемент меню = li (Элемент li находится внутри класса .hover_menu = hover_menu li)
3) отобразить подробные данные
    Прежде чем решить надо понять следующее:
    1) как скрыт тот или иной блок?
        ответ: 
            - если у него стоит display: none  
            то решением является его противоположность
            = display: block

            -если у него стоит opacity: 0
            то решение является его противоположность
            = opacity: 1

    Решение равно = display: block

    Общее решение:

    .hover_menu li:hover .dropdown_menu {
        display: block
    }
