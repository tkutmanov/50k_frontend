Правило CSS 

1) Правила подключения стилей прямо на странице
-используется тег style:

<style>
    Правила стилей
</style>

-данный кусок кода CSS должен располагаться перед закрывающим тегом </head>

2) Правило написания стилей:
    -начинается все с того, кому мы должны присвоить стили
        class  
        id  
        tag (тег)
        attribute  
    - мы должны открыть и закрыть фигурную скобку
        { }
    
        целостная картина равна:
        class {

        }  
        id {

        }  
        tag (тег) {

        }
        attribute{

        } 

3) Закрывающая фигурная скобка должна быть параллельно названию класса или же любого свойства
